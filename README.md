# 🤖 AI Chatbot with NLP & Memory

A web-based AI chatbot built using Python, Flask, and NLTK.
This chatbot processes user input using basic NLP techniques and maintains conversation history for contextual interaction.

---

## 🚀 Features

* Web-based chatbot interface (Flask)
* Natural Language Processing using NLTK
* Conversation memory tracking
* Modular and scalable architecture
* REST API for chatbot interaction

---

## 🛠️ Tech Stack

* Python
* Flask
* NLTK
* HTML/CSS/JavaScript

---

## 📦 Installation

```bash
git clone https://github.com/YOUR_USERNAME/ai-chatbot.git
cd ai-chatbot
pip install -r requirements.txt
```

---

## ▶️ Run the App

```bash
python app.py
```

Then open:
http://127.0.0.1:5000/

---

## 🧠 How It Works

* Tokenizes user input using NLTK
* Matches intent based on keywords
* Stores conversation history in memory
* Responds dynamically via Flask API
